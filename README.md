# CYORA 家庭平台｜策展頁

孕期與家庭情境策展頁面集合。

## 結構

```
cyora-curation/
├── shared/
│   ├── tokens.css       設計系統（顏色、字型、共用元件）
│   └── main.js          共用 JS（視差、白點、手風琴、切換器）
├── scenario-01-sleep/   情境一：懷孕後期，好好睡這件事
│   ├── index.html
│   └── images/
└── scenario-02-xxx/     情境二（待新增）
```

## 本地預覽

任意靜態伺服器皆可，例如：
```bash
npx serve .
```
